# Help Build NUOS Documentation

These documentation files are **linked directly from the NUOS web interface**.
Help build documentation for what users see and do in NUOS.

## What to contribute
- **Step-by-step guides** for common tasks
- **Interface explanations**: What each control and element does
- **Workflow examples**: How features work together
- **Best practices** for using NUOS effectively

## Do not document
- **Features not included in the standard distribution**

## How to contribute
1. Pick a feature you use regularly
2. Edit the corresponding file
3. Describe what users see and do
4. Submit a Pull Request

Thank you for helping improve NUOS documentation.
