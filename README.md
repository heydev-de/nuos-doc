# NUOS Web Interface Documentation

> Documentation for NUOS CMS & web IDE interface.
> These files are linked directly from the NUOS web interface.

## What is NUOS?
NUOS is a free, PHP/MySQL-based **CMS and web IDE** with a complete web interface for building and managing websites.

**Project Website:** https://nuos-cms.org

## Documentation Structure
- **Backend** – Management interfaces (editorial system, code editors, database, assets, ...)
- **Frontend** – Public modules (blog, forum, chat, user profiles, search, ...)
- **Desktop** – Workspace apps (calendar, contacts, email, messaging, ...)

**Only documents features available in the standard NUOS web interface.**
